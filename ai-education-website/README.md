# AI Education Website

## Overview
This project is a simple website dedicated to highlighting the importance of AI education in today's world. It features a responsive design with flippable cards that provide additional information about AI education when clicked.

## Project Structure
```
ai-education-website
├── src
│   ├── index.html          # Main entry point of the website
│   ├── why-ai-matters.html # Page detailing why AI education matters
│   ├── css
│   │   ├── style.css       # Main styles for the website
│   │   └── responsive.css   # Responsive styles for different screen sizes
│   ├── js
│   │   └── flippable-cards.js # JavaScript for card flipping functionality
│   └── images
│       ├── card-1.jpg     # Placeholder image for the first card
│       ├── card-2.jpg     # Placeholder image for the second card
│       └── card-3.jpg     # Placeholder image for the third card
├── README.md               # Documentation for the project
└── .gitignore              # Files and directories to ignore in version control
```

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `src/index.html` in your web browser to view the homepage.
4. Use the navigation to access the "Why AI Education Matters Now" page.

## Features
- Responsive design that adapts to various screen sizes.
- Flippable cards that reveal additional information about AI education statistics.
- Clean and modern layout with easy navigation.
