# AI Education Website

## Overview
This project is a simple website dedicated to highlighting the importance of AI education in today's world. It features a responsive design with flippable cards that provide additional information about AI education when clicked.

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `src/index.html` in your web browser to view the homepage.
4. Use the navigation to access the "Why AI Education Matters Now" page.

## Features
- Responsive design that adapts to various screen sizes.
- Flippable cards that reveal additional information about AI education statistics.
- Clean and modern layout with easy navigation.
