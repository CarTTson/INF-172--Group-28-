# Young and Hungry Podcast Website

A simple static website created for the **Young and Hungry** podcast project. This page showcases podcast information, statistics about AI education, features, and provides a form for visitors to apply to be guests on the show.

## Project Structure
```
index.html            # Main HTML page
css/
  └─ style.css        # Stylesheet for layout and appearance
images/               # Folder containing image assets
js/
  └─ flippable-cards.js  # JavaScript for interactive statistic cards
```

## How to Use
1. Clone or download the repository to your local machine.
2. Open `index.html` in a web browser (no build step required).
3. Edit the files as needed to customize content, styling, or functionality.

> This is a purely client‑side site; you can host it on any static web host like GitHub Pages, Netlify, or Vercel.

## Customization & Development
- **HTML**: Modify `index.html` to update sections, text, or navigation.
- **CSS**: Tweak `css/style.css` for styling changes and layout adjustments.
- **JavaScript**: Add or modify interactivity in `js/flippable-cards.js` or create new scripts.

## Tools & Requirements
- A modern web browser (Chrome, Firefox, Edge, etc.)
- No additional dependencies or build tools are required.
